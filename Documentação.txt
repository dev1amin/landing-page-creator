# Landing Creator Pro - Documentação

## 1. Visão Geral do Sistema

O Landing Creator Pro é uma plataforma especializada na criação e gerenciamento de landing pages dinâmicas, oferecendo três níveis de personalização (Básico, Avançado e Pro). O sistema permite que os clientes escolham templates predefinidos e personalizem seus conteúdos através de um painel administrativo intuitivo acessível via /admin.

### 1.1 Principais Características

- Sistema baseado em blocos modulares
- Painel administrativo intuitivo
- Templates responsivos e profissionais
- Hospedagem via Vercel
- Gerenciamento via GitHub
- Zero dependência de banco de dados
- Personalização em tempo real
- Sistema de upload de imagens integrado
- Personalização avançada de cores e estilos
- Suporte a múltiplos tipos de fontes

## 2. Sistema de Blocos

O Landing Creator Pro utiliza um sistema modular de blocos que permite a construção flexível de landing pages. Cada bloco representa uma seção específica da página e pode ser totalmente personalizado.

### 2.1 Blocos Disponíveis

1. **Hero (Cabeçalho Principal)**
   - Título principal
   - Subtítulo
   - Botão de chamada para ação (CTA)
   - Imagem principal
   - Personalização de cores
   - Alinhamento personalizável

2. **Features (Recursos)**
   - Título da seção
   - Múltiplos recursos
   - Ícones personalizáveis
   - Descrições
   - Layout em grid
   - Imagens por recurso

3. **Testimonials (Depoimentos)**
   - Título da seção
   - Depoimentos de clientes
   - Fotos dos clientes
   - Nome e cargo
   - Avaliação em estrelas (opcional)
   - Background personalizado

4. **Video (Vídeo)**
   - Título da seção
   - Incorporação de vídeo (YouTube/Vimeo)
   - Descrição
   - Thumbnail personalizada
   - Controles de reprodução

5. **About (Sobre)**
   - Título da seção
   - Texto descritivo
   - Imagem
   - Estatísticas (opcional)
   - Layout responsivo

6. **Services (Serviços)**
   - Título da seção
   - Lista de serviços
   - Ícones ou imagens
   - Preços (opcional)
   - Descrições
   - Links personalizados

7. **CTA (Chamada para Ação)**
   - Título chamativo
   - Texto persuasivo
   - Botão personalizado
   - Link configurável
   - Cores personalizáveis

8. **Pricing (Preços)**
   - Tabelas de preços
   - Recursos por plano
   - Botões de ação
   - Destaque para plano recomendado
   - Links personalizados
   - Estilos personalizáveis

9. **WhatsApp**
   - Botão flutuante
   - Número configurável
   - Mensagem predefinida
   - Posicionamento personalizado
   - Cores personalizáveis

10. **Instagram**
    - Botão de redirecionamento
    - Username configurável
    - Posicionamento personalizado
    - Cores personalizáveis
    - Link direto para perfil

11. **Facebook**
    - Botão de redirecionamento
    - URL da página configurável
    - Posicionamento personalizado
    - Cores personalizáveis
    - Link direto para página

12. **Custom (Personalizado)**
    - HTML personalizado
    - CSS personalizado
    - Integração com scripts externos
    - Estilização livre

13. **Footer (Rodapé)**
    - Logo personalizado
    - Links de navegação
    - Redes sociais
    - Informações de contato
    - Copyright
    - Cor da borda separadora personalizável
    - Texto adicional configurável

### 2.2 Personalização dos Blocos

Cada bloco pode ser personalizado em diferentes níveis, dependendo do plano:

**Básico:**
- Edição de textos
- Troca de imagens
- Links e botões
- Cores básicas

**Avançado:**
- Todas as opções do plano Básico
- Cores de fundo
- Cores dos textos
- Estilos dos botões
- Fontes personalizadas
- Espaçamentos

**Pro:**
- Todas as opções do plano Avançado
- Movimentação de blocos
- Adição/remoção de elementos
- Personalização avançada de layout
- HTML/CSS personalizado
- Integrações avançadas

## 5. Instruções de Uso

### 5.1 Primeiros Passos

1. Acesse o painel administrativo (/admin)
2. Escolha um template base
3. Personalize o conteúdo inicial
4. Configure as cores globais e fontes
5. Adicione os blocos desejados
6. Configure integrações sociais

### 5.2 Edição de Blocos

1. Clique no bloco para editar
2. Use o painel lateral para personalização
3. Visualize as alterações em tempo real
4. Configure cores e estilos
5. Ajuste espaçamentos e alinhamentos
6. Salve as modificações

### 5.3 Reordenação de Blocos (Pro)

1. Use as setas ↑↓ para mover blocos
2. Arraste e solte para reordenar
3. Visualize as alterações
4. Duplique blocos se necessário
5. Salve a nova organização

### 5.4 Gerenciamento de Imagens

1. Clique no botão de upload
2. Selecione ou arraste a imagem
3. Recorte se necessário
4. Otimização automática
5. Preview em tempo real
6. Confirme o upload
